1. How many kernels are listed under menuentry?

Open the GRUB configuration file:
	$ sudo less /boot/grub/grub.cfg
Search for menu entries:
/menuentry
Each line beginning with menuentry represents a boot option. Count the entries corresponding to Linux kernels. For example:

menuentry 'Ubuntu, with Linux 6.8.0-60-generic'
menuentry 'Ubuntu, with Linux 6.8.12'

The number on your system depends on how many kernels are installed.


2. Which is the default?
Check the GRUB default setting:
	$ grep "^GRUB_DEFAULT" /etc/default/grub

Common values are:
GRUB_DEFAULT=0
This means the first menuentry in the GRUB menu is booted by default.
You can also inspect:
grep "^set default" /boot/grub/grub.cfg
to see what GRUB ultimately uses as the default entry.


3. How can you manually choose which one to boot?
Reboot the system.
When the GRUB menu appears, use the arrow keys to select the desired kernel.
If the menu is hidden, hold Shift (BIOS systems) or press Esc repeatedly (UEFI systems) during boot to display it.
Select the kernel you want and press Enter.
If the kernel is listed under Advanced options for Ubuntu/Debian, select that submenu first, then choose the specific kernel version you want to boot.
